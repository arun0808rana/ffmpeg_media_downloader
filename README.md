# ffmpeg wrapper's wrapper

## Pre requisites

- ffmpeg 5.1.3-1
- nodejs >= 18

## Installation

```bash
npm i
```

## Start

```bash
node index.js --url <url> --filename <complete_path>
```

## Man

```bash
node index.js --help
Usage: index [options]

Options:
  -u, --url <url>            URL to download
  -f, --filename <filename>  Output filename
  -h, --help
```
